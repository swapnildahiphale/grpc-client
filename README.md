# grpc-client

Demo grpc-client app

## How to Build

Use the package [mvn](https://maven.apache.org) to build this project.

```bash
mvn install
```

## How to run

```bash
export GRPC_SERVER=localhost
# Optional
# export USER=my-user

java -jar grpc-client.jar
```
